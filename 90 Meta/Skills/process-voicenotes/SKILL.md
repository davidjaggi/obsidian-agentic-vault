# Skill: Process Voicenotes

**When to use**: A new transcript has landed in `40 Resources/Voicenotes/`
(synced there by the Voicenotes.ai Obsidian extension) and needs triage.

`40 Resources/Voicenotes/` is a sync target and landing zone, not a final
home - treat everything in it the same way as `00 Inbox/`: unprocessed
capture waiting to be routed.

**Steps**:
1. Read the transcript in full.
2. Decide where it belongs by actionability, same routing as inbox triage:
   - Has a deadline/outcome → `20 Projects/<project>/` (e.g. as a meeting
     note or appended to `LOG.md` if it's an update on an existing
     project).
   - An ongoing responsibility, no end date → `30 Areas/`.
   - Reference material with no immediate action → `40 Resources/`.
   - A distilled idea in your own words → `60 Notes/`, using
     `90 Meta/Templates/note.md` - rewrite in your own words, don't just
     paste the raw transcript.
3. Link the new note to at least one related existing note.
4. Delete the original transcript from `40 Resources/Voicenotes/` once
   triaged - it should not accumulate processed items.

**Depends on**: `90 Meta/Templates/note.md`
